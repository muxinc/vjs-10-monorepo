# FullscreenButton

A button component that toggles fullscreen mode for the media container.

## Features

- Automatically switches icons based on fullscreen state
- Works with browser Fullscreen API
- Falls back gracefully when fullscreen not supported
- Accessible keyboard navigation

## Usage

### React

```tsx
import { MediaProvider, Video, MediaContainer, FullscreenButton } from '@vjs-10/react';
import { FullscreenEnterIcon, FullscreenExitIcon } from '@vjs-10/react-icons';

function VideoPlayer() {
  return (
    <MediaProvider>
      <MediaContainer>
        <Video src="video.mp4" />
        <FullscreenButton className="fullscreen-button">
          <FullscreenEnterIcon className="fullscreen-enter-icon" />
          <FullscreenExitIcon className="fullscreen-exit-icon" />
        </FullscreenButton>
      </MediaContainer>
    </MediaProvider>
  );
}
```

### HTML

```html
<media-container>
  <video src="video.mp4"></video>
  <media-fullscreen-button class="fullscreen-button">
    <svg class="fullscreen-enter-icon"><!-- enter icon --></svg>
    <svg class="fullscreen-exit-icon"><!-- exit icon --></svg>
  </media-fullscreen-button>
</media-container>
```

## Styling

The FullscreenButton uses data attributes to reflect fullscreen state:

### Data Attributes

- `data-fullscreen` - Present when in fullscreen, absent when not

### CSS Example

```css
/* Position both icons in same grid cell */
.fullscreen-button {
  display: grid;
}

.fullscreen-enter-icon,
.fullscreen-exit-icon {
  grid-area: 1/1;
  transition: opacity 200ms;
}

/* Show enter icon when not fullscreen */
.fullscreen-button:not([data-fullscreen]) .fullscreen-enter-icon {
  opacity: 1;
}
.fullscreen-button:not([data-fullscreen]) .fullscreen-exit-icon {
  opacity: 0;
}

/* Show exit icon when fullscreen */
.fullscreen-button[data-fullscreen] .fullscreen-enter-icon {
  opacity: 0;
}
.fullscreen-button[data-fullscreen] .fullscreen-exit-icon {
  opacity: 1;
}
```

## Props

All standard button props are supported, plus:

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Button content (typically icons) |
| `className` | `string` | CSS class name |
| `target` | `RefObject<HTMLElement>` | Optional element to make fullscreen (defaults to MediaContainer) |

## Accessibility

- Automatically includes proper ARIA labels
- Keyboard accessible (Space/Enter)
- Announces fullscreen state changes to screen readers

## Browser Support

The FullscreenButton uses the standard Fullscreen API, which is supported in all modern browsers. On older browsers or contexts where fullscreen is not available, the button will be automatically hidden.
