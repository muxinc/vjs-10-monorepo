Welcome to Video.js v10! This guide will help you get up and running. In it, you'll make three key choices:

1. **Framework** - React or HTML (vanilla JavaScript)
2. **Skin** - The pre-built UI design you'll use, and how it's applied in your project
3. **Renderer** - The component that displays your media

## 1. Pick Your Framework

Video.js v10 supports both React and vanilla JavaScript/HTML through Web Components. 

First, select your preferred framework in the framework selector at the top of the <span class="hidden lg:inline">sidebar</span><span class="lg:hidden">page</span>.

Then, install the appropriate package:

<FrameworkCase frameworks={["react"]}>
```bash
npm install @videojs/react
```
</FrameworkCase>
<FrameworkCase frameworks={["html"]}>
```bash
npm install @videojs/html
```
</FrameworkCase>

## 2. Pick Your Skin

Skins are packaged sets of UI components and styles. Video.js v10 includes two built-in skins:

- **Frosted** - Modern glassy design with backdrop blur effects
- **Minimal** - Clean, simple design with straightforward controls

Learn more about skins <DocsLink slug="concepts/skins">in the guide â†’</DocsLink>

Currently, Video.js v10 uses CSS for styling these skins. More styling approaches (e.g, Tailwind CSS) are coming soon.

<FrameworkCase frameworks={["react"]}>
```tsx
import '@videojs/react/skins/frosted.css';
// or
import '@videojs/react/skins/minimal.css';
```
</FrameworkCase>
<FrameworkCase frameworks={["html"]}>
```ts
import '@videojs/html/skins/frosted';
// or
import '@videojs/html/skins/minimal';
```
</FrameworkCase>

## 3. Pick Your Renderer

<FrameworkCase frameworks={["react"]}>
Media renderers display your video and audio content. For now, Video.js v10 includes `<Video />`, which supports HLS streams.
</FrameworkCase>
<FrameworkCase frameworks={["html"]}>
Media renderers display your video and audio content. For now, you should use the native HTML5 `<video>` element.
</FrameworkCase>

More renderers for formats like DASH, and services like YouTube and Vimeo are planned.

## Basic Example

<FrameworkCase frameworks={["react"]}>
```tsx
import { MediaProvider, FrostedSkin, Video } from '@videojs/react';
import '@videojs/react/skins/frosted.css';

export default function App() {
  return (
    <MediaProvider>
      <FrostedSkin>
        <Video
          src="https://stream.mux.com/UZMwOY6MgmhFNXLbSFXAuPKlRPss5XNA.m3u8"
          poster="https://image.mux.com/UZMwOY6MgmhFNXLbSFXAuPKlRPss5XNA/thumbnail.webp"
          playsInline
        />
      </FrostedSkin>
    </MediaProvider>
  );
}
```
</FrameworkCase>
<FrameworkCase frameworks={["html"]}>
```ts
import { defineVjsPlayer } from '@videojs/html';
import '@videojs/html/skins/frosted';

defineVjsPlayer();

document.body.innerHTML = `
  <media-provider>
    <media-skin-frosted>
      <video
        slot="media"
        src="https://stream.mux.com/UZMwOY6MgmhFNXLbSFXAuPKlRPss5XNA.m3u8"
        poster="https://image.mux.com/UZMwOY6MgmhFNXLbSFXAuPKlRPss5XNA/thumbnail.webp"
        playsinline
      ></video>
    </media-skin-frosted>
  </media-provider>
`;
```
</FrameworkCase>

That's it! You now have a fully functional Video.js player. Go forth and play.
