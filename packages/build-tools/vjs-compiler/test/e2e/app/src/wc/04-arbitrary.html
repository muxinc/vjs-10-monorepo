Level 4: Arbitrary Values Skin (Web Component)

Testing: Arbitrary value support in Tailwind classes

Expected Features:

  • Arbitrary colors: Custom hex (#1da1f2) and rgba(0,0,0,0.3) values
  • Arbitrary sizing: clamp(3rem, 10vw, 5rem) for fluid button size
  • Arbitrary border-radius: Custom 12px radius
  • Arbitrary backdrop-blur: Custom 2px blur effect
  • Hover with arbitrary: Hover changes to custom color

Visual Checks:

  • Button: Twitter blue (#1da1f2) with 12px rounded corners
  • Button size: Fluid (3-5rem) based on viewport width
  • Overlay: Semi-transparent black (0.3 alpha) with subtle blur
  • Hover: Darker blue (#0d8ddb) on button hover
