Level 10: Named Groups - Web Component

Tests Tailwind's named group feature for nested group interactions

Test Instructions:

  • Named groups: Wrapper has group/root, controls container has group/controls
  • Root group hover: Hover over wrapper darkens overlay (group-hover/root:bg-[#000]/50)
  • Root group shadow: Hover over wrapper adds shadow to button (group-hover/root:shadow-[...])
  • Controls group ring: Hover over any button in controls adds ring to all buttons (group-hover/controls:ring-2)
  • Multiple groups: Both group/root and group/controls should work simultaneously
  • Console: Should have no errors
