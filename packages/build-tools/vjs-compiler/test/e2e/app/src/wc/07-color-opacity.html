Level 7: Color Opacity Modifiers Skin (Web Component)

Testing: Tailwind's slash opacity syntax for semi-transparent colors

Expected Features:

  • Overlay: Semi-transparent black background (bg-[#000]/50) with backdrop blur
  • Button: Twitter blue at 90% opacity (bg-[#1da1f2]/90)
  • Hover: Full opacity button on hover (hover:bg-[#1da1f2]/100)
  • Text: White text at 95% opacity (text-[#fff]/95)
  • Focus Ring: White ring at 30% opacity (ring-[#fff]/30)

CSS Technology:

  • Tailwind v4 uses color-mix(in oklab, color opacity%, transparent)
  • Modern CSS: Works in all browsers supporting color-mix() (2023+)
  • Better than rgba: Uses perceptually uniform OKLAB color space

Test Instructions:

  • Observe semi-transparent overlay over video
  • Hover over play button - should become fully opaque
  • Tab to button - should see semi-transparent focus ring
  • Click button - video should play/pause with smooth transitions
